### How to deploy
#### 1. make dir. & move to work-directory
        mkdir ~/flask-sample
        cd ~/flask-sample
#### 2. download this source
        git clone https://github.com/neotusca/flask-claude.git
#### 3. setting venv
        virtualenv venv
        source venv/bin/activate
#### 4. runnign flask-app
        cd flask-claude
        python3 app.py
#### 5. access test
        http://[server-public-ip]:5000 in web-browser
